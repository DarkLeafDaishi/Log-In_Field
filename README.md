# 🔐 Modern Login Page

A fully functional and responsive **Login Page** built using **HTML, CSS, and JavaScript**.  
This project demonstrates front-end form validation, UI design, and basic authentication logic.

---

## ✨ Features

• Responsive modern UI  
• Floating label input fields  
• Show / Hide password toggle  
• Form validation (email + password)  
• Demo authentication logic  
• Clickable profile image with fullscreen popup  
• Smooth animations and hover effects  
• Easy to connect to a backend (Firebase / Node.js)

---

## 🖼️ Preview

Login Page includes:
- Profile image popup
- Styled login card
- Error messages & success message

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox + Animations)
